
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Customize configuration

### 項目結構
```
├── build                      // 构建相关
├── src                        // 源代码
│   ├── api                    // 所有请求
│   ├── assets                 // 主题 字体等静态资源
│   ├── components             // 全局公用组件
│   ├── router                 // 路由
│   ├── store                  // 全局 store管理
│   ├── styles                 // 全局样式
│   ├── utils                  // 全局公用方法
│   ├── views                   // view
│   ├── App.vue                // 入口页面
│   ├── main.js                // 入口 加载组件 初始化等
├── babel.config.js            // babel 配置
├── .gitignore                 // git 忽略项
└── package.json               // package.json
```
